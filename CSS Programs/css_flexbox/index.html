CSS Flexbox:

Flexboxs allow you to set up a grid layout in your CSS stylesheet. Flexboxes adapt to screens size.

Parent div contains flex-container command.

Row and row-reverse align flex elements horizontally.

column and reverse column align flex verticle.

Justify-items command aligns items to left, center or right using (flex-start,flex-end and center inputs)

space-between and space-around can be used with justify-content to change space inbetween elements as page size changes.

Flex-wrap command can use wrap, wrap-refverse and no-wrap to change how elments wrap when screen size changes.

Align-items command can align items on site that are not even. commands for align-items are:(center, flex-end, flex-start)

Align-content command can change the way items align on a screen. commands for align-items are:(center, flex-end, flex-start, space-around, space-between)
