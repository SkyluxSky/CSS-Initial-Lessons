CSS Margins and Padding:

Padding is extra space on the inside, margins are for spacing outside of the element.
You can define padding for each side of an object.
You can also specify padding using the padding options with 4 values in a clockwise rotation starting from the top of the object.
Margins define outside space
You can define a margin for each side of an object.

You can define padding or a margin with a negative number.
