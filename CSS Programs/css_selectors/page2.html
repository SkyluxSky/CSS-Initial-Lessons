Selectors allow you to select any element in your HTML file. Two most common selectors are class and id's

* asteriks in css style sheet allows you to select all the elements in the html document.

You can select multiple elements in css by using comma's (ex h1, p) will select all p's and h1's in the doc.

You can select values in lists by using the ul > li command.

You can select an element that comes after another by using a + sign. ( p + h1 ) styles the h1.

You can select elements based on there html attributes. (ex. selecting an href would be represented by [href] a list item can be selected by typing li[object])

You can select any element that has a title wit a certain phrase of chaaracters (ex. [title*="ban"] = all elements with the characters "ban")

Input elements can have attributes given to them by typing input:focus or check in stylesheet.

Input elements can have attributes after they are enabled (ex. input[type="checkbox"]:checked changes checkbox when checked.)

:hover changes attributes when you hover over elements

You can change attributes of child (ex. ul > li:____-child)
