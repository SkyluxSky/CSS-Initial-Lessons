CSS Classes and Id's:

Classes and Id's make it easier to reuse CSS stylesheets. Classes and Id's are generally the same and allow us to define predefined CSS styling and use them on our HTML elements.

Id's can be used if you want to style an element on an individual page of HTML. Each element can only have 1 id.

You can give an HTML element as many classes as you want. You can construct classes in css documents and use them in HTML elements. Multiple classes can be used per element.
