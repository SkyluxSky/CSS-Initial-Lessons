CSS Advanced Styling: Shadows, Opacity and Border Radius

Opacity controls the transparency of an element is measured between 0.0-1.0

Text Shadow coontrols shadow of the text /// requires two px values for horizontal and verticle shadows /// you can also give text shadow a color // blur attributes are placed after the horizontal and vertical shadows attribute and are measured in pixels.

Box shadow creates a shadow for an image

Border radius controls how square the border of an element is. 50% for border radius will create a circle
